# 동사 업그레이드 묶음 — 후속 패스 (2026-06-10)

## 요약
> 06-09 코어 시스템 재기획 §2.3.1 "나머지 동사 후속 패스"를 이어, 동사별 **기본형 + 업그레이드 옵션 묶음(유저 1택)** 과 **밸런스 초안**을 정리했다. ⑤복제·⑥임시방편 묶음은 이 세션에서 직접 합의(✅). **①~④ 묶음은 직전 대화가 컨텍스트 압축으로 유실되어 06-09 패턴 기반 재구성안(🔶)으로 기록.** 산출물 = [[superpowers/specs/2026-06-10-동사-업그레이드-묶음-밸런스초안-design]].

## ⚠ 유실 경위 (다음 세션 필독)

- 이 세션 진행 중 컨텍스트가 요약·압축되며, **동사 ①복구·②개조·③연성·④방출의 업그레이드 옵션 묶음 세부가 AI 메모리에서 소실**됐다(요약 표에 "앞서 확정" 플래그만 잔존).
- 이 창에 **온전히 남은 것 = ⑤복제·⑥임시방편 묶음 + 6종 기본형 정의**뿐.
- 따라서 ①~④ 묶음은 **날조 대신 재구성안(🔶)** 으로 명시 기록. 원안과 다를 수 있음 → **사용자 검증 필수.**

## 🔴 최우선 미결 — 로스터 정합 (6종 ↔ 8종)
- 이번 표 = **6종**(복구·개조·연성·방출·복제·임시방편).
- 승인 스펙 §2.3 = **8종**(교체·수리·투척·덧대기·과부하·재가동·복제·임시방편).
- ⑤·⑥만 일치. ①복구≈수리, ④방출≈투척/과부하(추정). ②개조·③연성=신규. **교체·덧대기·과부하·재가동 누락.**
- **결정 전까지 06-09 스펙 비파괴**(별도 후속 문서로 보존). 대체로 결정되면 §2.3/§2.3.1 병합.

## 확정 (✅)
- **6종 기본형** — 스펙 §1 표.
- **⑤ 복제** — 강화 사본 / 다중 복제(2) / 영속 사본. (공명 제외)
- **⑥ 임시방편** — 즉석 affix / 공격형 임시 파츠(위력↑+처치 AP 환급) / 긴급 일괄 충전(빈 슬롯 수만큼).

## 재구성 (🔶 검토 요)
- **①복구** — 과수복 / 광역 복구 / 응급 안정화.
- **②개조** — 정밀(선택 부여) / 중첩(상한+1) / 교리(키스톤 우대).
- **③연성** — 등급+1 / affix 계승 / 부위 지정.
- **④방출** — 광역 / 형질(파츠 affix 연동) / 연쇄(처치 시 추가).
- **밸런스 §3** — AP+재료 단일축 가설값. 플레이테스트 튜닝 전제.
