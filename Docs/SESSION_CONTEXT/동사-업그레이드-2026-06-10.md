## "6종 표" 정리 결과 (검증 완료)

| 6종 명칭 | 검증 | 처리 |
|---------|------|------|
| 복구 | 수리 계열, 수리가 덧대기 흡수 | **삭제** → 덧대기 |
| 개조 | 발언 기록 0 (AI 명명). 실제 의도 = 덧대기 시 재료 affix 이전 | **삭제** → 덧대기 업그레이드 |
| 연성 | 근거 0 (옛 매치 = "유연성" 부분문자열) | **폐기** |
| 방출 | 8종 대응 없음 | **폐기** |

## 확정 (✅)
- **수리 → 덧대기 흡수** (전투 중 내구도 복구 = 덧대기). 수리 독립 동사 아님.
- **투척 = 투척** (불변). **교체 = §2.2 상시 기본기**(슬롯 ✕). 과부하·재가동·복제·임시방편 불변.
- **덧대기 업그레이드:** 덧대기에 사용한 재료 파츠의 **affix 중 1개를 0~15% 확률로 결과 파츠에 이전.**
- **투척 업그레이드(§2.3.1):** 두 갈래 택1 — ⓐ 장착 파츠도 / ⓑ 인벤토리 멀쩡 파츠도 던지기 (둘 다 ✕).
- **복제 업그레이드:** 강화 사본 / 다중(2) / 영속 사본. (공명 제외) — 이전 세션 합의, 재확인 권장.
- **임시방편 업그레이드:** 즉석 affix / 공격형(위력↑+처치 AP환급) / 긴급 일괄(빈 슬롯 수만큼) — 이전 세션 합의, 재확인 권장.

## 미정

- 🔴 복제·임시방편 묶음 재확인.
- ❌ 과부하·재가동 업그레이드 미정의 (후속 패스).
