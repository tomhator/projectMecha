# SESSION_CONTEXT — Agent Guide

이 폴더는 AI 에이전트가 대화 세션의 작업 내용을 정리·저장하는 공간이다.
세션이 종료되거나 컨텍스트가 일정 수준에 도달하면 에이전트가 직접 파일을 생성한다.

---

## 저장 트리거

다음 중 하나에 해당하면 즉시 저장한다:

- 대화 메시지 교환 **8~10회 이상** 진행
- 코드 작업·기획 논의 등 실질적인 작업이 다수 완료
- 긴 파일 읽기/편집이 반복되어 컨텍스트 소비가 큰 경우

저장 전 사용자에게 반드시 알린다:
> "컨텍스트 20% 초과로 세션 내용을 저장합니다."

---

## 파일명 규칙

```
{대화주제}.md
{대화주제}-{YYYY-MM-DD}.md   ← 동일 주제를 여러 날 작업할 때
```

- `{대화주제}`: 세션의 핵심 주제를 한국어 또는 영어로 간결하게 명명
- 예: `메카-전투-시스템-설계.md`, `부품-장착-UI-구현.md`, `턴제-로직-리팩토링.md`

---

## 문서 템플릿

```markdown
---
date: YYYY-MM-DD
tags: [ProjectMecha, {관련태그1}, {관련태그2}]
status: in-progress | completed
topic: {대화주제 요약}
---

# {대화주제}

## 요약
> 이 세션에서 다룬 내용을 2~3문장으로 요약

## 작업 내용

### 논의된 사항
- 항목별로 정리

### 구현/변경된 파일
- `경로/파일명.gd` — 변경 내용 한 줄 요약

### 결정 사항
- 중요한 설계 결정이나 방향 선택

## 미완료 / 다음 세션에서 할 일
- [ ] TODO 항목 1
- [ ] TODO 항목 2

## 관련 문서
- [[Docs/GameDesignDocument]]
- [[Docs/WorkNote/YYYY-MM-DD]]
```

---

## 저장 규칙

1. 저장 후 파일 경로를 사용자에게 알린다.
2. 같은 주제의 파일이 이미 존재하면 **내용을 추가(append)** 하되, 날짜를 구분하여 이어붙인다.
3. 저장 완료 후에도 현재 세션을 계속 진행한다.
