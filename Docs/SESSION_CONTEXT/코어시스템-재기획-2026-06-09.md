# 코어 시스템 재기획 — 코어/어빌리티/교리/키스톤 (2026-06-09 저녁)

## 요약
> "코어 성장과 파츠 재기획" 중 **코어 기둥**을 대화로 전면 재기획했다. 아침 boon 진단에서 드러난 갭(2교리 vs 3축, 코어 어빌리티 분류 얇음)을 푸는 토대 작업. **코어**=단일 주인공·런중 고정·메타 노드 성장 / **코어 어빌리티**=동사 8종 3슬롯 / **교리**=형식 태그축(태운다·살린다) / **키스톤**=노드 점수 합산+문턱+기본공격 타이브레이커 / **메타**=규칙 비트는 키스톤. 산출물 = [[superpowers/specs/2026-06-09-코어시스템-재기획-design]]. **파츠 기둥(덱+affix 태그)은 다음 세션 이월.**

## 작업 내용

### 확정 사항 (대화 합의 — 상세는 스펙)
- **코어(본체)** — 단일 주인공(스토리 상수), 절대 교체·해금 없음. 런 중 고정(HP/하중/AP/슬롯). 성장=메타로 *단일 코어의 노드 트리* 확장. 이중 주인공: 스토리=코어 / 게임플레이=파츠("파일럿은 무대, 파츠는 배우").
- **코어 어빌리티** — 파츠 조작 동사 그 자체. 출격 전 **3슬롯 로드아웃**(런 중 고정), 메타로 로스터 확장 + 슬롯 1→2→3(+후반 4번째).
- **상시 기본기**(콘솔 밖) — 기본공격(출격 전 **노드 택**, 교리 보유=타이브레이커)·긴급 장착(파손 빈 슬롯 복구). cf. 교체=멀쩡한 파츠 자발 교체.
- **동사 8종 → 3택** — 교체·수리·투척·덧대·과부하·재가동·복제·임시방편. (백로그: 분해·강탈.)
- **휘발 3종**(재가동·복제·임시방편) — 전투 한정, 끝나면 소멸 ↔ **작업대=영구 공급**(시간축 분리: 계획 정비 vs 급박 임기응변). **재가동=화장**(50% 부팅 → 내구 소진 시 완전 소멸).
- **교리** — 상자 아닌 **축/지향점**(태운다/살린다, 확장형). 형식 추적 (가), backbone=태그. (C) 노드선언 + 런누적.
- **키스톤** — 코어 설계=노드 빌드, 노드마다 교리 점수. ①합산(기본공격 제외) ②동점→기본공격 타이브레이커 ③선두가 문턱↑→강한 키스톤 / 미달→제너럴리스트.
- **메타 성장** — 스탯 뻥튀기 ❌ → **규칙 비트는 키스톤 + 트레이드오프** ⭕ (예: "재가동 파츠 소멸 안 함, 대신 ○○ 포기"). 코어 업그레이드+교리 패시브 통일.

### 핵심 합의의 결 (왜)
- "파츠가 주인공" → 런 파워는 파츠가, 코어는 그릇.
- 데메크 네로식 소비 전투 → 휘발 동사로 *즉흥* 공급, 작업대로 *영구* 공급.
- "교리에 안 가둔다" → 동사·노드 자유 조합, 키스톤은 *결과로* 결정.
- 메타 = 전투 spec §1.4 가드레일("트레이드오프가 깊어짐")과 정합.

### 이어진 결정 (2026-06-09 후속 패스 — 동사 디테일)
- **공통 비용 축 = AP + 재료 (단일축).** 새 자원 0, 기존 코드 정합. 가드레일 "공짜 금지" 충족.
- **시작 1슬롯.** (메타로 1→2→3→4.)
- **투척:** 기본=파손 파츠만 → 메타 **두 갈래 택1**(ⓐ장착 파츠도 / ⓑ멀쩡 파츠도). 둘 다 불가.
- **수리:** 기본=파손 파츠 소모 → 업글로 장착·인벤 멀쩡 파츠도 소모(파츠로 파츠 살림).
- **휘발 산출물(복제 사본·임시방편):** 즉시 장착 → 빈 슬롯 없으면 **큐 최상위**로.
- **통일 패턴 확정:** "동사 = 기본형 + 메타 업그레이드(일부 두 갈래 택1 트레이드오프)" = §5 메타 형식과 동일. 나머지 동사도 같은 틀로 후속 확정.
- 스펙 반영: §2.1 비용축·슬롯, §2.3.1 동사 기본형→업글, §2.4 산출물 배치, §5 메타 예시, §8 미결 갱신.

### 이어진 결정 (2026-06-09 후속 — 기본공격·코어 업그레이드·노드 구조)
- **기본공격:** 현 3종 유지 + **교리 태그 매핑**(입자 광선=태운다 / 간이 방벽=살린다 / 속사 사격=중립). 키스톤 동점 타이브레이커. (스펙 §2.2.1)
- **코어 업그레이드 노드 = 규칙 + 스탯(대가 필수) 혼합.** 순수 +스탯 금지, 스탯 주면 반드시 대가. 옛 5레벨 선형 +스탯·옛 T5 3축 캡스톤 교리 **폐기**. §1 "성장=메타" ↔ §5 "스탯 뻥튀기 ❌" 모순 해소. (스펙 §5.1)
- **노드 빌드 구조 = 평면 노드망.** 옛 **5티어 게이팅·노드 레벨(1~5) 폐지.** 진행 곡선=재화 예산·해금 순서. (스펙 §6)
- **옛 어빌리티 트리(AbilityTreeSystem) 전면 재기획 필요** — 5티어×3축·5레벨·외장슬롯·`core_research.cfg` 전부 옛 3축 모델 → 별도 대수술 패스(§7·§10).
- 미결: 재화 예산·해금 곡선 수치, **코어 외장↔노드 매핑 재설계**(티어 폐지 여파), 개별 노드 로스터·수치·대가.

### boon 의존성 해소
- 아침 진단(boon-설계-2026-06-09)의 **갭①(2교리 vs 3축)·갭③(분류 얇음·수리 누락)을 설계로 해소.** 교리=형식 2교리 태그축 확정, 동사 8종에 수리 포함.
- boon 남은 선행 = **affix 교리 태그(파츠 기둥)**. boon = 런 한정 교리 태그로 키스톤/정렬 방향 증폭.

## ▶ 다음 세션 재개 지침 (파츠 기둥 착수 시)

> 0단계 — 코어 스펙 + 이 문서 다시 읽기. "파츠가 주인공" 유지.

1. **덱 구조 확정** — 부위 슬롯 4 + 인벤토리 큐. **고정 진열형(A) vs 드로우형(B) 미결**(내 추천 A: 전투 중 운보다 판단). 동사(교체/긴급장착/투척)가 큐를 건드림.
2. **affix 전면 재설계** — 현 풀 파기 → **교리 태그(태운다/살린다) 붙은 시너지 글루**로. 파츠=덱 카드, 직관적 스킬+affix.
3. **파츠 등급/롤/획득 재검토** — 덱빌딩 손맛이 *전투 밖(획득·분해·제작·affix)*에서 나오게.
4. **정합 업데이트** — 작업대 "코어 어빌리티 티어 → 제작 affix 상한"을 노드/키스톤 기준으로 재정의(코어 스펙 §7). 전투 spec §3.4 "[교체]→[긴급 장착]".
5. 이후 → **던전 이벤트(조우·상자·boon).** boon은 affix 태그 확정 후.

## 미결 (코어 스펙 §8)
- 동사 효과 수치 + 공통 비용 축(AP/고철/횟수).
- 문턱 최소 점수·교리 점수 스케일.
- 노드 트리 구조·예산·해금 곡선·슬롯 시작 개수(1슬롯 빈약 우려).
- 백로그 동사(분해·강탈) 추가 시점.
