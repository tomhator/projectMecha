# 맵화면-재설계 (+ 코어 전환 문서 정리)

## 요약
> 오전: 새 코어("파츠가 주인공") 전환의 문서 부채를 정리(작업대 GDD 동기화·검증기준 nV1~nV7 확정·스크랩→고철 통합). 오후: 던전 맵 선택 화면을 대화 협업으로 재설계 — StS식 연결 그래프 + "경로 = 보급 전략" + 방 분포(전투밀도·부위 특화·교리 노드+런 boon). **boon 세부는 내일.**

## 작업 내용

### 논의된 사항 (맵 화면 — 전부 대화로 합의)
- **구조:** 3번(보급 프레이밍) + 1번(StS식) 결합. "경로 선택 = 보급 전략"이 게임 주제("파츠가 주인공")와 직결.
- **StS 그래프:** 최대 4칸 폭, 갈래 합쳐졌다 갈라짐, **엣지 제약**(연결 노드로만 이동·하나 고르면 다른 길 포기), 전체 맵 처음부터 공개.
- **노드:** 종류 아이콘 + 예상 보상 등급(점). **작업대는 노드 아님 → [공급] 탭으로 상시 접근.**
- **노드 디테일:** 호버 **및 포커스** 툴팁(옆 패널 제거 — 맵 가독성 우선). 우측 **희미한 층 눈금**(숫자/텍스트 없음, 현재 줄만 강조).
- **전역 보급 readout**(맵 구석 상시): 부위별 멀쩡 파츠 갯수(0인 부위 경고색)·고철 ⚙✦·파손(부위 무관 합산)·인벤. 공용 상단바는 안 건드림.
- **방 분포:** 전투밀도 축(강행군=해체공 색·고철/파츠 income / 비축=정비공 색·크레딧), 부위 특화=확정 아님 **확률↑+희소**(다리 더 드물게), 교리 노드=가끔 끼는 특수 전투 → 클리어 시 **런-한정 boon**(하데스식).

### 결정 사항
- 작업대 = 맵 노드 아님, prep `[공급]` 탭으로 상시. (GDD §2.2·§2.3 정합은 미결)
- 스크랩 → **고철(커먼/레어)** 로 통합 (재화 종류 최소화, "시스템 심화 금지" 부합).
- **"V6"은 죽은 게이트** — 검증 트리거는 nV1~nV7, 동결은 *구현*만 게이트(설계는 자유).
- 교리 boon = **런-한정**(하데스 신 boon), 영구 교리 성장(재화 투자)과 **별개 레이어**(시간축 다름).
