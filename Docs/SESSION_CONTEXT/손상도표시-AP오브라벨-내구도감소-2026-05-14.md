# 손상도 표시 (조립·전투) + AP 오브 라벨 + 내구도 감소

## 요약
> 조립 씬 소켓과 전투 HUD 파츠 목록에 ■/□ 형식 손상도 블록을 추가했다. 행동력 오브 옆에 "행동력" 라벨을 붙여 의미를 명확히 했다. 스킬을 사용할 때마다 해당 파츠의 `durability`가 1씩 감소하며 전투 HUD가 즉시 갱신된다.

---

## 작업 내용

### 논의된 사항

1. **손상도 미표시 문제**
   - 조립 씬 `PartSocketUI`: 장착된 파츠에 `⚠` 태그만 있었고 ■/□ 블록이 없었음.
   - 전투 HUD `CombatUI._update_parts_display()`: 파츠 이름만 표시, 손상도 전혀 없었음.

2. **AP 오브 혼동**
   - 조립 씬 좌하단 녹색 오브가 "행동력"을 뜻하는지 플레이어가 알 수 없었음.
   - `_update_ap_orbs()`에서 오브를 생성할 때 첫 번째 자식으로 "행동력" 라벨을 함께 추가하는 방식으로 해결.

3. **스킬 사용 시 내구도 감소 미구현**
   - `MechaEntity.use_skill()`에서 파츠 스킬(`_skill_to_part[skill] != null`)을 쓸 때 `durability`가 감소하지 않았음.
   - 코어 기본 스킬은 내구도 영향 없음.

### 구현/변경된 파일

- **`Scripts/Autoload/EventBus.gd`**
  - `signal part_durability_changed(part: PartsData)` 추가.

- **`Scenes/Entities/MechaEntity.gd`**
  - `use_skill()` 끝에 내구도 감소 로직 추가:
    ```gdscript
    var part: PartsData = _skill_to_part.get(skill)
    if part != null and part.durability > 0:
        part.durability -= 1
        EventBus.part_durability_changed.emit(part)
        if part.durability == 0:
            print("  > [파츠 파괴] %s" % part.parts_name)
    ```

- **`Scenes/Assembly/PartSocketUI.gd`**
  - `var _durability_label: Label` 멤버 변수 추가.
  - `_build_ui()`: `_skill_label` 아래에 `_durability_label` 생성 (font 10, 우측 정렬).
  - `_refresh_display()`:
    - 빈 슬롯이면 `_durability_label.text = ""`
    - 장착 시 `"■".repeat(durability) + "□".repeat(max_durability - durability)` 표시
    - 색상: 녹색(정상) / 주황(손상, `is_worn()`) / 빨강(파괴, `is_broken()`)
    - `⚠` 태그 제거, 파츠 이름 색으로 상태 표현으로 통일.

- **`Scenes/UI/CombatUI.gd`**
  - `_ready()`: `EventBus.part_durability_changed.connect(_on_part_durability_changed)` 추가.
  - `_on_part_durability_changed(_part)`: `_update_parts_display()` 재호출.
  - `_update_parts_display()`:
    - 슬롯명 영문(`ARM_L`) → 한국어(`왼팔`, `오른팔`, `등`, `다리`) 로 변환하는 `_slot_korean()` 헬퍼 추가.
    - 각 파츠 줄에 손상도 블록 추가: `"%s: %s  %s" % [슬롯명, 파츠명, dur_str]`
    - 파츠 파괴(빨강) / 손상(주황) / 빈 슬롯(회색) 색상 코딩.

- **`Scenes/Assembly/AssemblyScene.gd`**
  - `_update_ap_orbs()`: 오브 생성 전 "행동력" `Label`을 `_ap_orbs_row`의 첫 자식으로 추가.

### 결정 사항

- **내구도 감소 단위**: 스킬 1회 사용 = durability 1 감소. 현재는 전투 종료 후 자동 회복 없음 (미구현).
- **파츠 파괴 처리**: `durability == 0`이 되면 콘솔 출력만. 실제 전투 효과 패널티는 `MechaEntity._output_mult()`의 `is_worn()` 체크(`×0.7`)로 이미 반영됨.
- **코어 스킬 내구도**: `_skill_to_part[skill] == null` → 내구도 소모 없음.
