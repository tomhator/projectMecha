- **V1 밴드 이탈**: 파손 비용·드롭 등급 곡선 재조정(GDD §4.5/§2.3). 연속 2회 실패 → 교체 비용 모델 자체 재검토(파손 대신 수리비 등).
- **V2/V5**: 드롭 테이블 "같은 부위 충돌 확률" 조정. 첫 런 한정 2층 같은-부위 드롭 보장 점검.
- **V3/V4**: 조립/교체 UI 비교 표시 개선. 3회 실패 → "교체 전후 핵심 수치 1줄 요약" 강제 표시.
- **V6**: 핵심 루프 기획(코어루프 3요소)으로 복귀해 재진단.
