## 0. "6종 표" 정리 결과 + 유실 복구 경위

직전 세션 "6종 표"의 명칭(복구·개조·연성·방출)을 정본 8종·git 이력과 대조한 결과:

| 6종 명칭 | 검증 결과 | 처리 |
|---------|----------|------|
| 복구 | 수리 계열. 수리가 덧대기에 흡수 → 별도 불필요 | **삭제** → 덧대기 |
| 개조 | "개조"는 발언 기록 없음(AI 명명). 실제 의도 = 덧대기 시 재료 affix 이전 | **삭제** → 덧대기 업그레이드 |
| 연성 | 근거 0 (옛 git 매치는 "유연성" 부분문자열) | **폐기** |
| 방출 | 8종 대응 없음 | **폐기** |

### ⚠ 유실 복구 (중요)

2026-06-09에 확정한 **나머지 동사 업그레이드 표(§2.3.2)** 가 커밋 `e56c190`에 있었으나,
`716085a`("남은 동사·어빌리티 트리는 대화로 재논의")로 **revert**되며 현재 문서에서 사라졌었다.
→ §2에 **복구**해 다시 기록한다. (과부하·재가동·교체·복제·임시방편·덧대 업그레이드 방향)

### 정정
- **교체 = 슬롯 동사**(중립/회전, 재료 없음·AP만). 상시 기본기는 **긴급 장착**(파손 슬롯 자동 복구)이다. (이전 정리에서 둘을 혼동 → 수정.)

### 8종 대비 확정 변경
- **수리 → 덧대기 흡수** (전투 중 내구도 복구는 덧대기).
- **투척 = 투척** (불변). 과부하·재가동·복제·임시방편 = 불변.

## 1. 동사 8종 — 내용

| # | 동사 | 내용(기본형) | 교리 | 지속성 | 슬롯 |
|---|------|------------|------|--------|------|
| 1 | **교체** | 멀쩡 장착 파츠를 큐로 보내고 큐 파츠 장착. **재료 없음·AP만** | 중립/회전 | 영구 대상 | ○ |
| 2 | **수리** | 내구도 복구 | 살린다 | 영구 | **덧대기에 흡수**(독립 ✕) |
| 3 | **덧대기** | 파손/재료 파츠 소모 → 보강·임시 방호판(쉴드) **+ (수리 흡수) 내구도 복구** | 살린다 | 영구 | ○ |
| 4 | **투척** | 파츠를 적에게 던져 피해 + 소비 | 태운다 | 영구(소비) | ○ |
| 5 | **과부하** | 한 파츠 스킬 출력 폭주. 사용 후 파괴 | 태운다 | 영구(가속 소모) | ○ |
| 6 | **재가동** | 파괴된 파츠를 내구 50%로 되살림 | 집착→소비 | **휘발/화장** | ○ |
| 7 | **복제** | 파츠 1개의 **1회용** 사본 생성·장착 | 버스트(태운다) | **휘발** | ○ |
| 8 | **임시방편** | 능력 없이 "때리기"만 가진 임시 파츠 생성·장착 | 즉흥/중립 | **휘발** | ○ |

- **긴급 장착 = 상시 기본기**(파손으로 빈 슬롯 자동 복구) — 동사 슬롯과 별개(06-09 §2.2). 교체는 *멀쩡 파츠* 능동 전환이라 별개 슬롯 동사.
- **휘발 3종**(재가동·복제·임시방편) 규칙 = 06-09 §2.4.
- 수리가 덧대기에 흡수되므로, **슬롯에 넣는 동사 = 7종**(교체·덧대기·투척·과부하·재가동·복제·임시방편).

## 2. 동사별 업그레이드 방향

> **복구 표 = 2026-06-09 확정**(e56c190 revert분). 비용 전부 **AP + 재료**(§2.1), 효과 수치·갈래형 확정값은 미결.
> 06-10 세션에서 일부 동사를 갱신 → 충돌 셀은 🔴로 표시(아래 §2.1 재논의).

| 동사 | 기본형 | 업그레이드 방향 (택1, 복구 06-09) | 06-10 갱신 |
|------|--------|-----------------------------------|------------|
| **교체** | 재료 없음·AP만 | ⓐ 교체 AP 0(무료 회전) / ⓑ 새로 낀 파츠 첫 사용 강화 | — |
| **덧대기** | 파손 파츠 소모 → 임시 방호판(쉴드) | ⓐ 멀쩡 파츠도 덧대(더 큰 방호) / ⓑ 방호 지속화(임시→지속) | 🔴 **+ affix 0~15% 이전** · 수리 흡수(내구 복구) |
| **투척** | 파손 파츠만 던짐 | ⓐ 장착 파츠도 / ⓑ 인벤토리 멀쩡 파츠도 (둘 다 ✕, §2.3.1) | — |
| **과부하** | 폭주 → 사용 후 파괴 | ⓐ 파괴 대신 **파손 상태로**(재활용) / ⓑ 피해 더↑ 대신 **코어 반동** | — |
| **재가동** | 50% 부활·휘발/화장 | ⓐ 부활 내구↑(50→75%) / ⓑ 대상 +1(연쇄). *영구화=코어 업글 키스톤* | — |
| **복제** | 1회용 사본·휘발 | ⓐ 사본 2회용 / ⓑ 사본도 과부하 가능 | 🔴 06-10 안: 강화 사본 / 다중(2) / 영속 |
| **임시방편** | 때리기 임시 파츠·휘발 | ⓐ 약한 능력 1개 부여 / ⓑ 2개 생성 | 🔴 06-10 안: 즉석 affix / 공격형 / 긴급 일괄 |

### 2.1 충돌 셀 — 재논의 필요 🔴

세 동사는 **06-09 확정안**과 **06-10 세션안**이 다르다. 어느 쪽을 채택할지(또는 병합) 결정 필요:

- **덧대기** — 06-09: 쉴드 기반(멀쩡 파츠 덧대 / 지속화). 06-10: 내구 복구(수리 흡수) + affix 0~15% 이전.
  → 기본형이 "쉴드"인가 "내구 복구"인가부터. 업그레이드도 4개 후보(멀쩡/지속/affix이전/연료확장) 중 묶음 구성 결정.
- **복제** — 06-09: 2회용 / 과부하 가능. 06-10: 강화 / 다중(2) / 영속. (병합 가능? 예: 다중 + 과부하가능 + 영속)
- **임시방편** — 06-09: 약능력 1개 / 2개 생성. 06-10: 즉석 affix / 공격형 / 긴급 일괄. ("약능력 1개"≈"즉석 affix", "2개"≈"긴급 일괄" → 병합 여지)

## 3. 밸런스 — 보류

- 수치는 내용·업그레이드 확정 후 별도 패스 + headless/플레이테스트.
- **불변 원칙:** AP + 재료 단일축(§2.1) · "공짜 금지" · 휘발 동사 AP 무겁게 · 업그레이드는 수치 inflation 아닌 **운용 분기**(§5.1).

## 4. 미결 / Open Questions

- 🔴 **충돌 3동사 재논의**(§2.1) — 덧대기·복제·임시방편: 06-09안 ↔ 06-10안 채택/병합.
- 🔴 **덧대기 기본형 정체** — 쉴드 vs 내구 복구(수리 흡수) vs 둘 다.
- ❌ **수치·갈래형 확정값** — 06-09 표는 *방향*만, 효과 수치 미결(§8).
- **덧대기 affix 이전 ↔ 작업대 정합** — affix 조작이 작업대(영구 공급)와 겹침. 시간축 분리(§2.4).
- **8종 교리 태그 매핑** 은 위 표에 1차 기재(교체=중립/회전, 덧대기=살린다, 투척·과부하·복제=태운다, 재가동=집착→소비, 임시방편=즉흥/중립).
- 확정분 → 06-09 §2.3/§2.3.1/§2.3.2(복구) 병합.
