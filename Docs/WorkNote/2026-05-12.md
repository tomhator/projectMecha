## 결정 사항

- affix는 이름에 포함 안 함 → 카드 UI 아이콘으로 표시
- 등급 = affix 개수 / 드롭 희귀도 = 파츠별 개별 가중치
- 파츠별 affix 가중치 풀 보유 (파츠마다 잘 붙는 affix 경향 있음)
- 총기류는 ARM_L / ARM_R 양쪽 드롭 풀에 등장 (좌우 공용)
- 코어 이름(Vanguard, Striker, Bulwark)과 파츠 이름 중복 금지
