## 작업 0. 기존 스킬 target_slot 초기화

**이유:** 아래 스킬들에 이전 디자인 잔재로 `target_slot` 값이 남아 있음.  
작업 4 구현 후 저격 로직이 활성화되면 **일반 적도 즉시 저격 스킬 사용자**가 됨.  
일반 병종에는 저격 스킬이 없어야 하므로, 구현 전에 미리 초기화.

**수정 대상 (5개 파일)**

| 파일 | 현재 target_slot | 변경 후 |
|------|----------------|---------|
| `Resources/Skills/skill_rapid_fire.tres` | `1` (ARM_R) | `-1` (NONE) |
| `Resources/Skills/skill_heavy_punch.tres` | `0` (ARM_L) | `-1` (NONE) |
| `Resources/Skills/skill_scatter_shot.tres` | `3` (LEG) | `-1` (NONE) |
| `Resources/Skills/skill_cannon_shot.tres` | `2` (BACK) | `-1` (NONE) |
| `Resources/Skills/skill_colossus_strike.tres` | `2` (BACK) | `-1` (NONE) |

각 파일에서 `target_slot = <숫자>` 줄을 `target_slot = -1` 로 교체.

## 작업 1. `get_available_skills()` — 파괴 파츠 스킬 제외

**파일:** `Scenes/Entities/MechaEntity.gd` (line 20~23)

### 문제

PASSIVE 타입만 걸러내고 파괴 여부는 무시함.  
모든 스킬이 파괴 파츠 소속일 때 자동 턴 종료가 안 되는 버그.

### 수정

```gdscript
func get_available_skills() -> Array[SkillData]:
    return available_skills.filter(func(s: SkillData) -> bool:
        if s.skill_type == SkillData.SkillType.PASSIVE:
            return false
        var part: PartsData = _skill_to_part.get(s)
        if part != null and part.is_broken():
            return false
        return true
    )
```

> `_skill_to_part.get(s)` 가 `null`이면 코어 스킬 → 파괴 체크 자동 생략.

## 작업 4. `EnemyEntity.execute_actions()` — 슬롯 내구도 감소

**파일:** `Scenes/Entities/EnemyEntity.gd` (line 71)

### 수정 — execute_actions

```gdscript
func execute_actions(target: Node) -> void:
    for action: SkillData in next_actions:
        print("[%s] '%s' 사용" % [enemy_name, action.skill_name])
        if action.skill_damage > 0.0:
            target.take_damage(action.skill_damage * attack_multiplier)
            if action.target_slot != SkillData.TargetSlot.NONE:
                _apply_snipe_to_slot(target, action.target_slot)
        if action.skill_defense > 0.0:
            _heal_shield(action.skill_defense)
        if action.skill_heal > 0.0:
            _heal_hp(action.skill_heal)
            print("  > [%s] HP +%.0f" % [enemy_name, action.skill_heal])
        EventBus.skill_used.emit(self, action)
    decide_next_actions()


func _apply_snipe_to_slot(target: Node, slot_index: int) -> void:
    if not target.has_method("get_part_at_slot"):
        return
    var part: PartsData = target.get_part_at_slot(slot_index)
    if part == null:
        print("  > [저격] 슬롯 %d 비어 있음 — 내구도 효과 무효" % slot_index)
        return
    if part.is_broken():
        print("  > [저격] %s 이미 파괴됨 — 내구도 효과 무효" % part.parts_name)
        return
    part.durability = maxi(part.durability - 1, 0)
    EventBus.part_durability_changed.emit(part)
    print("  > [저격] %s 내구도 감소 → %d" % [part.parts_name, part.durability])
    if part.is_broken():
        print("  > [파츠 파괴] %s" % part.parts_name)
```

### 추가 — MechaEntity.get_part_at_slot()

## 작업 5. 빈 슬롯 저격 예고 하이라이트 숨김

**파일:** `Scenes/Entities/EnemyEntity.gd` (line 113)

### 수정

```gdscript
func _resolve_snipe_preview_slot() -> int:
    for action: SkillData in next_actions:
        if action == null:
            continue
        if action.skill_target != SkillData.SkillTarget.ENEMY:
            continue
        if action.target_slot == SNIPE_SLOT_NONE:
            continue
        if not _player_has_part_at(action.target_slot):
            continue
        return action.target_slot
    return SNIPE_SLOT_NONE


func _player_has_part_at(slot_index: int) -> bool:
    for slot: CoreData.CoreSlot in GameState.equipped_parts:
        if int(slot) == slot_index:
            return GameState.equipped_parts[slot] != null
    return false
```

> 파괴된 파츠는 슬롯에 존재하므로 `true` 반환 → 예고 표시됨 (의도된 동작).

## 작업 6. `CollectorArmEntity` 클래스 작성

**파일:** `Scenes/Entities/CollectorArmEntity.gd` (신규)

수집가의 팔 하나를 표현하는 경량 엔티티. `EnemyEntity`를 상속.

```gdscript
extends EnemyEntity
class_name CollectorArmEntity

# 이 팔이 강탈한 플레이어 파츠 (null이면 보스 기본 팔)
var stolen_from_part: PartsData = null

static func create_default(arm_type: String, arm_skill: SkillData) -> CollectorArmEntity:
    var arm := CollectorArmEntity.new()
    arm.enemy_name = arm_type          # 예: "압착 팔", "드릴 팔"
    arm.enemy_max_hp = 45.0
    arm.enemy_max_shield = 0.0
    arm.attack_multiplier = 1.3        # 수집가 배율 공유
    arm.enemy_action_count = 2
    arm.skills = [arm_skill]
    return arm


static func create_from_stolen(part: PartsData) -> CollectorArmEntity:
    var arm := CollectorArmEntity.new()
    arm.stolen_from_part = part
    arm.enemy_name = "강탈된 " + part.parts_name
    arm.enemy_max_hp = 45.0
    arm.enemy_max_shield = 0.0
    arm.attack_multiplier = 1.3
    arm.enemy_action_count = 2
    # 파츠의 첫 번째 스킬을 그대로 사용
    if not part.skills.is_empty():
        arm.skills = [part.skills[0]]
    return arm
```

> `PartsData`에 `skills: Array[SkillData]` 필드 존재 여부 확인 후 필요 시 수정.

## 작업 7. `BossCollectorEntity` 코어 클래스

**파일:** `Scenes/Entities/BossCollectorEntity.gd` (신규)

```gdscript
extends EnemyEntity
class_name BossCollectorEntity

const MAX_ARMS: int = 4

var active_arms: Array[CollectorArmEntity] = []
# 팔 탈취 횟수 제한 (선택적 밸런스 옵션 — 현재는 무제한)
# var theft_count: int = 0

# 보스 전용 스킬 참조 (setup_from_data 전 preload 또는 외부 주입)
@export var skill_emp_shock: SkillData = null
@export var skill_arm_theft_ref: SkillData = null  # 탈취 판정용 더미 SkillData

# 팔 5종 스킬 풀 (CollectorArmEntity 생성에 사용)
@export var arm_skill_pool: Array[SkillData] = []  # 압착/스윕/집게/드릴/방어 스킬


func setup() -> void:
    super.setup()
    _spawn_arms(MAX_ARMS)


func _spawn_arms(count: int) -> void:
    for i in count:
        var skill: SkillData = arm_skill_pool[randi() % arm_skill_pool.size()]
        var arm := CollectorArmEntity.create_default(_arm_name_for(skill), skill)
        arm.setup()
        active_arms.append(arm)
        # TurnManager에 등록 요청
        EventBus.boss_arm_spawned.emit(arm)


func _arm_name_for(skill: SkillData) -> String:
    # 스킬 이름으로 팔 종류 이름 결정 (간단한 매핑)
    match skill.skill_name:
        "유압 압착": return "압착 팔"
        "광역 스윕": return "스윕 팔"
        "연속 집기": return "집게 팔"
        "드릴 돌격": return "드릴 팔"
        "방어막":    return "방어 팔"
        _:           return "강탈 팔"


func decide_next_actions() -> void:
    # 팔 전멸 시 재수집 트리거 (다음 플레이어 턴 시작 전 즉시 실행)
    if active_arms.is_empty():
        _trigger_recollection()
    # 팔 탈취 조건 충족 시 후보에 추가
    var theft_available: bool = active_arms.size() < MAX_ARMS
    # 기존 decide_next_actions 로직 (skills 배열 기반 랜덤 선택)
    # + skill_emp_shock 포함
    # + theft_available이면 skill_arm_theft_ref 포함
    _decide_with_theft_option(theft_available)


func _decide_with_theft_option(include_theft: bool) -> void:
    next_actions.clear()
    var pool: Array[SkillData] = skills.duplicate()  # 고철 흡수, 몸통 박치기, EMP 포함
    if include_theft and skill_arm_theft_ref != null:
        pool.append(skill_arm_theft_ref)
    var ap_remaining: int = enemy_action_count
    var candidates: Array[SkillData] = pool.filter(
        func(s: SkillData) -> bool: return s.skill_action_cost <= ap_remaining
    )
    while not candidates.is_empty():
        var chosen: SkillData = candidates[randi() % candidates.size()]
        next_actions.append(chosen)
        ap_remaining -= chosen.skill_action_cost
        candidates = pool.filter(
            func(s: SkillData) -> bool: return s.skill_action_cost <= ap_remaining
        )


func _trigger_recollection() -> void:
    print("[수집가] 팔 전멸 — 재수집 시작")
    _spawn_arms(MAX_ARMS)


func execute_actions(target: Node) -> void:
    for action: SkillData in next_actions:
        if action == skill_arm_theft_ref:
            _apply_arm_theft(target)
        else:
            # 부모 클래스 단일 스킬 실행 로직 재활용
            _execute_single_action(action, target)
    decide_next_actions()


func _execute_single_action(action: SkillData, target: Node) -> void:
    print("[%s] '%s' 사용" % [enemy_name, action.skill_name])
    if action.skill_damage > 0.0:
        target.take_damage(action.skill_damage * attack_multiplier)
        if action.target_slot != SkillData.TargetSlot.NONE:
            _apply_snipe_to_slot(target, action.target_slot)
    if action.skill_defense > 0.0:
        _heal_shield(action.skill_defense)
    if action.skill_heal > 0.0:
        _heal_hp(action.skill_heal)
    EventBus.skill_used.emit(self, action)


func _apply_arm_theft(target: Node) -> void:
    if not target.has_method("steal_part_at_slot"):
        return
    # ARM_L(0) / ARM_R(1) 중 장착된 슬롯 랜덤 선택
    var candidates: Array[int] = []
    for slot_idx in [0, 1]:   # ARM_L, ARM_R
        if target.get_part_at_slot(slot_idx) != null:
            candidates.append(slot_idx)
    if candidates.is_empty():
        print("[수집가] 팔 탈취 실패 — 플레이어 ARM 슬롯 비어 있음. 소타격 적용.")
        target.take_damage(10.0 * attack_multiplier)
        return
    var chosen_slot: int = candidates[randi() % candidates.size()]
    var stolen: PartsData = target.steal_part_at_slot(chosen_slot)
    if stolen == null:
        return
    print("[수집가] '%s' 강탈 완료!" % stolen.parts_name)
    var new_arm := CollectorArmEntity.create_from_stolen(stolen)
    new_arm.setup()
    active_arms.append(new_arm)
    EventBus.boss_arm_spawned.emit(new_arm)
```

> **arm_skill_pool** `.tres` 설정: 보스 리소스 파일에서 압착/스윕/집게/드릴/방어 팔 스킬 5종을 배열에 추가.

### 8-2. MechaEntity에 함수 추가

**파일:** `Scenes/Entities/MechaEntity.gd`

```gdscript
## 지정 슬롯의 파츠를 강탈당함. 슬롯을 비우고 스킬 목록에서 제거.
## 반환값: 강탈된 파츠 (없으면 null)
func steal_part_at_slot(slot_index: int) -> PartsData:
    for slot: CoreData.CoreSlot in GameState.equipped_parts:
        if int(slot) != slot_index:
            continue
        var part: PartsData = GameState.equipped_parts[slot]
        if part == null:
            return null
        # 슬롯 비우기
        GameState.equipped_parts[slot] = null
        # 스킬 목록에서 해당 파츠 스킬 제거
        for skill in part.skills:
            available_skills.erase(skill)
            _skill_to_part.erase(skill)
        EventBus.part_stolen.emit(part, slot_index)
        print("[MechaEntity] '%s' 강탈됨 — 슬롯 %d 비워짐" % [part.parts_name, slot_index])
        return part
    return null
```

### 9-2. 승리 조건 수정 — 보스 코어 격파 = 승리

현재 `_check_combat_end()`는 **모든** 적이 defeated일 때만 승리 처리.  
수집가 전투에서는 팔이 남아있어도 코어가 격파되면 승리.

```gdscript
func _check_combat_end() -> bool:
    if GameState.current_hp <= 0.0:
        _end_combat(false)
        return true
    # 보스 코어 격파 우선 체크 (팔 엔티티 남아있어도 승리)
    for enemy: EnemyEntity in enemies:
        if enemy is BossCollectorEntity and enemy.is_defeated():
            _end_combat(true)
            return true
    # 일반 전투: 모든 적 격파
    var all_defeated: bool = enemies.all(
        func(e: EnemyEntity) -> bool: return e.is_defeated()
    )
    if all_defeated:
        _end_combat(true)
        return true
    return false
```

> `BossCollectorEntity`가 아닌 일반 전투는 기존 동작 그대로.

## 주의 사항

1. **`CoreData.CoreSlot` vs `SkillData.TargetSlot` 정수값**  
   `_apply_snipe_to_slot()`·`steal_part_at_slot()`에서 정수 비교.  
   `CoreData.gd` 열어서 `CoreSlot` enum 순서(ARM_L/ARM_R/BACK/LEG)가 0/1/2/3인지 먼저 확인.

2. **`PartsData.skills` 필드 존재 여부**  
   `CollectorArmEntity.create_from_stolen()`에서 `part.skills[0]` 접근.  
   `PartsData.gd` 에서 `skills: Array[SkillData]` 필드 확인 후 없으면 추가.

3. **팔 탈취 더미 SkillData (`skill_arm_theft_ref`)**  
   팔 탈취는 실제 `SkillData`로 표현하되 `execute_actions()`에서 분기 처리.  
   `skill_arm_theft_ref`의 `skill_action_cost = 3`, `skill_damage = 0`으로 설정.  
   행동 예고 UI에 "팔 탈취" 텍스트가 표시되도록 `skill_name = "팔 탈취"`.

4. **EventBus `boss_arm_spawned` 시그널 타입**  
   `CollectorArmEntity`는 `EnemyEntity` 서브클래스이므로  
   시그널 타입을 `EnemyEntity`로 선언해도 무방.
