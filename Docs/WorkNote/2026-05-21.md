## 특이사항 / 결정 사항

- 트리 포인트는 런당 3pt 고정. 한 분기 완주(코어 스킬 언락)에 3pt 전부 필요 → 분산 or 집중 선택 강요
- 노드 해제(환불) 허용: 상위 노드 미선택 시에만 가능
- `core_base.tres`는 `duplicate()`로 매 런마다 독립 복사본 생성 → 트리 스탯 수정이 다음 런에 영향 없음
- 코어 스킬 3종은 기존 스킬 재활용: `skill_railgun_pierce`, `skill_iron_shield`, `skill_assault_leap`
- 던전 중 트리 노드 찍는 기능은 1차 미구현 (거점에서만)
