# 파츠 카탈로그 (Parts Catalog)

> [!info] 관련 문서
> - 시스템 명세: [[PartsSystem]]
> - Affix 상세: [[AffixSystem]]

> [!warning] 작업 전 읽기
> 이 문서는 **고유 파츠 30종**의 상세 기획 문서다.  
> 각 파츠의 스킬·수치·affix 풀·드롭 가중치를 여기서 설계한다.  
> .tres 파일 생성은 이 문서 확정 후 진행.

---

## 작성 규칙

### 파츠 항목 구조

| 항목 | 설명 |
|------|------|
| **파일명** | `part_{슬롯}_{식별자}.tres` |
| **고유명** | 파츠의 정체성을 나타내는 이름. 드롭 시 접두사가 앞에 붙음 |
| **스킬** | 이 파츠가 제공하는 스킬 이름 + 한 줄 설명 |
| **base 수치** | 기준 수치 (실제 값 = base × stat_multiplier) |
| **드롭 가중치** | 상대 가중치. 높을수록 자주 등장. 전체 합산 기준으로 확률 계산 |
| **affix 풀** | 이 파츠에서 등장 가능한 affix ID 목록 (→ [[AffixSystem]] 참고) |
| **affix 가중치** | affix 풀과 1:1 대응. 높을수록 해당 affix가 자주 붙음 |

### 드롭 가중치 기준

| 가중치 범위 | 체감 희귀도 |
|-----------|-----------|
| 15~20 | 흔함 (자주 등장) |
| 8~14 | 보통 |
| 3~7 | 드묾 |
| 1~2 | 매우 드묾 |
